# Financial Dashboard

## Summary
A financial dashboard that visualizes various data in tables and candlestick graphs, displaying crucial information in a clean and aesthetically appealing way.

## Features
- Supports **Dark/Light Mode**
- **Responsive** design for both mobile and desktop
- Multiple **viewing modes** for different pages

## Pages

### Home Page
- Displays consistent trade data in a user-friendly layout.
![Home Page](/src/assets/SC_dark_HomePage.png)

### Table Overview
- Shows **four tables**:
  - Credit
  - Holdings
  - Risk
  - Transactions
- Consists **four viewing modes (3 for mobile) **:
    - Single View with Tabs
    - Side by Side
    - All Grids (2x2 Grid)
    - All Grids (Scroll)

    ![Single Mobile View](/src/assets/SC_dark_mobile_TableOverviewSingle.png)
    ![Side by Side Mobile View](/src/assets/SC_dark_side-by-side.png)
    ![Grid](/src/assets/SC_dark_TableOverviewGrid.png)

### Candlestick Page
- Displays various **security tickers** with a dropdown for selection.
- Visualizes data using **candlestick charts**.
![Candlestick](/src/assets/SC_light_candlestick.png)
